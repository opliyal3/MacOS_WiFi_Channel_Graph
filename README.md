# MacOS_WiFi_Channel_Graph
A  python script using MacOS comment(airport) for viewing your wifi channel usage vs. signal strength, and draws a graph.

# Depends
* pandas
* numpy
* matplotlib

# Usage

```pytohn
python3 WiFi_channel.py <root_password>
```
